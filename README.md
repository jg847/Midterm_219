## What is it?
*Grounded Noves* is an Next.js application with an AI Chatbot that helps you find job listings, rentals, and create budget plans. Users are able to change their locations and help them find jobs listings and rentals around a user-input radius. The site also contains a *Story* tab where users can interact with the slider to calculate their rent burden. A *Resource* tab is available so users can select a location and manually find job listings, rental options, and more support.
